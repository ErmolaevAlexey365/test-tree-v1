# Running the app
Pull from repository.
Production branch: **main**.
QA branch: **QA**.
Develop branch: **develop**.

### Required
Node `v16` or higher

### Run
```
npm install
npm run start
```

### Build
Available env's - prod, qa
```
npm install
npm run build:[ENV]
```
